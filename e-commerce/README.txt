				E-COMMERCE

GETTING STARTED
Τοποθετήστε τον φάκελο e-commerce στο directory του server.
*Ο server πρέπει να υποστηρίζει php και mysql.

Ο φάκελος e-commerce περιέχει πέντε υποφακέλους :
• τον css και τον js που είναι αρχεία που χρησιμοποιεί η bootstrap,
• τον Fonts, που περιέχει κάποια fonts ποθ χρησιμοποιούνται στην σελιδα,
• τον img που αποθηκεύονται οι εικόνες τον αγγελιών,
• και τον moderator που αφορά την ιστοσελίδα του διαχειριστή.
Επίσης περιέχει και τα αρχεία php και το style.css.

Ο φάκελος css περιέχει τον core κώδικα της bootstrap που είναι απαραίτητος ενώ
το αρχείο style.css περιέχει τις δικές μας προσθήκες όσον αφο΄ρα την εμφάνιση του site.


Ο φάκελος moderator, πιο αναλυτικά, περιέχει τον φάκελο images, που περιέχει μέσα τις εικόνες
για την σελίδα του διαχειριστή, και τα αρχεία php της σελίδας.
